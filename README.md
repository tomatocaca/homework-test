# 无人机送货系统 (Drone Delivery System)

一个基于 C++ 的控制台应用程序，模拟无人机物流配送系统的完整业务流程，支持多角色操作（管理员、配送员、普通用户）。

## 功能概览

### 管理员
- 添加无人机（轻型/中型/重型）
- 查看所有无人机信息
- 查看所有订单信息
- 查看系统统计报告（订单状态统计、货物类型统计、无人机使用率）
- 查看当日无人机配送记录
- 用户封禁管理（手动解封）

### 配送员
- 查看待处理订单
- 分配订单给空闲无人机（自动匹配载重和货物类型）
- 执行配送
- 查看已完成订单记录

### 普通用户
- 下单（即日配送或预约1-3天内配送）
- 查看我的订单
- 确认收货
- 取消订单（每周取消达3次自动封禁2周）

## 系统架构

```
配送地图（线性拓扑）：
  A(天马公寓) — B(德智公寓) — C(综合楼) — D(东方红广场)
  相邻站点配送时长：10 分钟
```

### 无人机等级

| 等级 | 载重上限 | 可运送货物 | 特殊功能 |
|------|---------|-----------|---------|
| 轻型 | ≤5 kg | 普通货物 | 无 |
| 中型 | ≤20 kg | 普通/生鲜/药品 | 温控系统(2°C-8°C) |
| 重型 | ≤50 kg | 全品类 | 温控系统 + 保险锁 |

### 订单状态流转

```
Pending → Assigned → Delivering → Delivered → Received
  ↓                                                       
  +→ Cancelled (任意非配送中状态可取消)
```

## 编译与运行

```bash
# 使用 g++ 编译
g++ -std=c++17 -o drone_delivery_system.exe drone_delivery_system.cpp

# 运行
drone_delivery_system.exe
```

## 环境要求

- 支持 C++17 标准的编译器（g++ / MSVC / Clang）
- Windows 系统（自动切换 UTF-8 编码，防止中文乱码）
